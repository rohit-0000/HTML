HR TAG

Creates a horizontal line across the page.

Used to seperate content sections.

It doesn't have an end tag (hr tag self-closing)

BR TAG

Insert a line break in the text
Useful for breaking lines of text within paragraphs or other block elements.
Also self-closing

BOLD TAG

b : Makes text bold
Example :- This Text is bold

ITALIC TAG

i : Italicizes text
Example :- This text is italic

UNDERLINE TAG

u : Underline the text.
Example :- This text is underlined.

BIG TAG

Increases the font size of the enclosed text slightly larger than surrounding text.
Example :- This is big

SMALL TAG

Decreases the font size of the enclosed text slightly smaller than the surrounding text.
Example :- This text is small.

SUBSCRIPT

To display text as subscript, you use the sub tag. This is commonly used in mathematical or chemical notations.
Example :- H2O

SUPERSCRIPT

To display text as superscript, you use the sup tag. This is often used for exponents or ordinal indicators.
Example :- E = mc2

PRE TAG

The pre tag in HTML is used to display preformatted text. Text inside this tag is displayed in a fixed-width font (usually Courier), and both spaces and line breaks are preserved. This is useful for displaying code blocks, or when you want to display text exactly as it's typed without HTML rendering collapsing spaces or ignoring line breaks.

It will print as it as given

Example :-

        #include
        using namespace std;
        int main()
        {
            cout<<"HELLO WORD \n";
            return 0;
        }
